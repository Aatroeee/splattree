SplatTree

Voxel-Based Multi-Resolution Decoding for Feed-Forward 3D Gaussian Generation
Ran Gong · Zhehong Ren · Yueyu Hu · Tingyu Fan · Yao Wang
New York University
Paper Poster Code coming soon
SplatTree uses a hierarchical 3D pipeline for structured, multi-resolution Gaussian generation.

Abstract

Geometry vision transformers such as VGGT jointly predict depth, camera poses, and rich visual features from multi-view images in a single forward pass. However, existing VGGT-based 3D Gaussian Splatting methods decode Gaussians independently in 2D image space, leaving the spatial context in 3D underused.

SplatTree bridges this gap by lifting VGGT's geometry-grounded features into a unified 3D voxel representation and decoding Gaussian attributes with sparse 3D convolutions. It further introduces a multi-resolution decoder that generates Gaussians at multiple voxel scales, enabling coarse-to-fine rendering under different computation or bandwidth budgets.
