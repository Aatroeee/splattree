SplatTree

Voxel-Based Multi-Resolution Decoding for Feed-Forward 3D Gaussian Generation
Ran Gong · Zhehong Ren · Yueyu Hu · Tingyu Fan · Yao Wang
New York University
Paper Poster Code coming soon

Abstract

SplatTree is a feed-forward 3D Gaussian generation framework that moves Gaussian decoding from 2D image space into 3D voxel space. It lifts geometry-aware features from VGGT into a unified voxel representation and decodes Gaussian attributes with sparse 3D convolutions.

The method further introduces a multi-resolution decoder that generates Gaussians at different voxel scales. Coarse levels provide a lightweight approximation, while finer levels can be activated when more computation or storage is available.
